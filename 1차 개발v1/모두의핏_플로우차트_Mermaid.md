# 모두의 핏 앱 플로우차트 (Mermaid)

## 1. 전체 앱 구조 플로우

```mermaid
graph TD
    A[앱 실행] --> B[로그인 메인 페이지]
    B --> C[통합 로그인]
    B --> D[카카오 로그인]
    B --> E[비밀번호 찾기]
    B --> F[회원가입]
    B --> G[파트너 등록]
    
    E --> E1[이름 입력]
    E1 --> E2[전화번호 인증]
    E2 --> E3[비밀번호 재설정]
    E3 --> E4[재설정 완료]
    
    F --> F1[기본정보 입력]
    F1 --> F2[약관 동의]
    F2 --> F3[사용자 설정]
    F3 --> F4[가입 완료]
    
    G --> G1[기본정보 입력]
    G1 --> G2[약관 동의]
    G2 --> G3[올피커 설정]
    G3 --> G4[파트너 타입 선택]
    
    G4 --> H[지도자 등록]
    G4 --> I[시설 운영자 등록]
    G4 --> J[통합 등록]
    
    H --> H1[등록 완료]
    I --> I1[등록 완료]
    J --> J1[등록 완료]
    
    C --> K[메인 앱]
    D --> K
    F4 --> K
    H1 --> K
    I1 --> K
    J1 --> K
```

## 2. 파트너 등록 상세 플로우

```mermaid
graph TD
    A[파트너 등록 시작] --> B[기본정보 입력]
    B --> B1[이름 입력]
    B --> B2[이메일/전화번호 입력]
    B --> B3[비밀번호 설정]
    
    B3 --> C[약관 동의]
    C --> D[올피커 설정]
    D --> D1[닉네임 설정]
    D --> D2[생년월일 입력]
    D --> D3[성별 선택]
    D --> D4[거주지역 설정]
    
    D4 --> E{파트너 타입 선택}
    
    E -->|지도자| F[지도자 등록 프로세스]
    E -->|시설 운영자| G[시설 운영자 등록 프로세스]
    E -->|통합| H[통합 등록 프로세스]
    
    F --> F1[기본 정보 입력<br/>+ 자격 정보 업로드]
    F1 --> F2[포트폴리오 작성]
    F2 --> F3[소속 시설 등록]
    F3 --> F4[등록 완료]
    
    G --> G1[기본 정보 입력<br/>+ 증빙서류 업로드]
    G1 --> G2[시설 등록]
    G2 --> G3[운영 시간 설정]
    G3 --> G4[등록 완료]
    
    H --> H1[지도자 정보 입력<br/>+ 자격증 업로드]
    H1 --> H2[시설 정보 입력<br/>+ 사업자등록증 업로드]
    H2 --> H3[통합 등록 완료]
```

## 3. 메인 앱 플로우

```mermaid
graph TD
    A[메인 앱] --> B[홈 탭]
    A --> C[지도 탭]
    A --> D[마이페이지 탭]
    
    B --> B1[현재 위치 표시]
    B --> B2[시설 검색창]
    B --> B3[운동 분야 선택]
    B --> B4[근처 인기 시설]
    
    B1 --> B11[위치 변경]
    B3 --> B31[구기 종목 선택]
    B31 --> B32[시설 리스트]
    B32 --> B33[시설 상세]
    B33 --> B34[지도자 탭]
    
    D --> D1[마이페이지 메인]
    D1 --> D2[프로필 수정]
    D1 --> D3[계정 변경]
    D1 --> D4[더보기]
    
    D3 --> D31[전화번호 변경]
    D3 --> D32[이메일 변경]
    D3 --> D33[비밀번호 변경]
    
    D4 --> D41[약관 정책]
    D4 --> D42[계정 관리]
    
    D41 --> D411[서비스 이용 약관]
    D41 --> D412[개인정보 처리방침]
    D41 --> D413[위치기반 서비스 약관]
    
    D42 --> D421[로그아웃 확인]
    D42 --> D422[회원 탈퇴 확인]
```

## 4. 지도자 등록 상세 플로우

```mermaid
graph TD
    A[지도자 등록 선택] --> B[기본 정보 입력<br/>+ 자격 정보 업로드]
    B --> B1{유료 전환 안내 팝업}
    B1 -->|확인| B2[개인 사업자번호]
    B1 -->|취소| B
    
    B2 --> B3[사업자등록증 업로드]
    B3 --> B4[자격증 업로드]
    
    B4 --> C[포트폴리오 작성]
    C --> C1[프로필 이미지]
    C --> C2[한줄 소개]
    C --> C3[전문 분야]
    C --> C4[활동 사진]
    C --> C5[SNS 계정]
    
    C5 --> D[소속 시설 등록]
    D --> D1[시설명 입력]
    D1 --> D2[시설 주소 입력]
    D2 --> D3[위치 확인]
    
    D3 --> E[등록 완료]
    E --> E1[승인 대기 안내]
```

## 5. 시설 운영자 등록 상세 플로우

```mermaid
graph TD
    A[시설 운영자 등록 선택] --> B[기본 정보 입력<br/>+ 증빙서류 업로드]
    B --> B1{유료 전환 안내 팝업}
    B1 -->|확인| B2[대표자명]
    B1 -->|취소| B
    
    B2 --> B3[사업자번호]
    B3 --> B4[사업자등록증 업로드]
    B4 --> B5[시설 사진 업로드]
    
    B5 --> C[시설 등록]
    C --> C1[시설명 입력]
    C --> C2[시설 주소 입력]
    C --> C3[시설 유형 선택]
    C --> C4[운동 종목 선택]
    
    C4 --> D[운영 시간 설정]
    D --> E[등록 완료]
    E --> E1[승인 대기 안내]
```

## 6. 통합 등록 상세 플로우

```mermaid
graph TD
    A[통합 등록 선택] --> B[지도자 정보 입력<br/>+ 자격증 업로드]
    B --> B1[지도자 기본 정보]
    B --> B2[개인 사업자번호]
    B --> B3[사업자등록증 업로드]
    B --> B4[자격증 업로드]
    
    B4 --> C[시설 정보 입력<br/>+ 사업자등록증 업로드]
    C --> C1[시설 기본 정보]
    C --> C2[대표자명]
    C --> C3[사업자번호]
    C --> C4[사업자등록증 업로드]
    C --> C5[시설 사진 업로드]
    
    C5 --> D[통합 등록 완료]
    D --> D1[두 권한 모두 승인 대기]
```

## 사용 방법

1. 위의 Mermaid 코드를 복사
2. [Mermaid Live Editor](https://mermaid.live/)에 붙여넣기
3. 자동으로 플로우차트가 생성됨
4. PNG, SVG 등으로 다운로드 가능

## 주요 특징

- **파트너 등록**: 일반 사용자 가입부터 시작하여 올피커 설정까지 완료 후 역할 선택
- **3가지 선택지**: 지도자, 시설 운영자, 통합 등록
- **유료 전환 안내**: 지도자 등록 시 팝업으로 사전 안내
- **기존 정보 활용**: 이미 입력한 정보를 파트너 등록 시 재사용
- **승인 프로세스**: 모든 파트너 등록은 관리자 승인 후 활성화
