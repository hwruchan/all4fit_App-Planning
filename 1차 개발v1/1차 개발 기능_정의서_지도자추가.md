# 모두의 핏 앱 기능 정의서 - 지도자 기능 추가

## 1. 문서 개요

### 1.1 프로젝트 정보
- **프로젝트명**: 모두의 핏 (All4Fit) 앱
- **버전**: 2.1
- **작성일**: 2025년 10월
- **대상 플랫폼**: iOS, Android

### 1.2 문서 목적
본 문서는 기존 1차 개발 기능 정의서에 지도자 관련 기능을 추가한 확장 버전입니다. 홈 화면에서 지도자 탭을 통한 인기 지도자 조회 및 상세 정보 확인 기능을 정의합니다.

### 1.3 추가 개발 범위
- **지도자 조회**: 홈 화면에서 지도자 탭을 통한 인기 지도자 목록 조회
- **지도자 상세**: 지도자 선택 시 상세 정보 확인
- **시설과 동일한 구조**: 시설 조회와 유사한 UI/UX 구조 적용

## 2. 전체 앱 구조 (지도자 기능 추가)

### 2.1 앱 아키텍처 (지도자 기능 포함)
```
모두의 핏 앱 (ROOT)
├── 인증 영역 (AUTH) - 비로그인 상태
│   ├── 로그인 메인 (LOGIN_MAIN)
│   ├── 지도자 회원가입 (INSTRUCTOR_SIGNUP)
│   └── 시설 운영자 회원가입 (FACILITY_OPERATOR_SIGNUP)
├── 메인 앱 (MAIN) - 로그인 상태
│   ├── 홈 탭 (HOME_TAB)
│   │   ├── 현재 위치 표시 (HOME_LOCATION)
│   │   ├── 시설 검색창 (HOME_SEARCH)
│   │   ├── 시설 탭 (FACILITY_TAB)
│   │   │   ├── 운동 분야 선택 (HOME_SPORTS)
│   │   │   │   └── 구기 종목 선택 (HOME_SPORTS_DETAIL)
│   │   │   │       └── 시설 리스트 (FACILITY_LIST)
│   │   │   │           └── 시설 상세 (FACILITY_DETAIL)
│   │   │   │               └── 지도자 탭 (INSTRUCTOR_TAB)
│   │   │   └── 내 주변 인기 시설 (HOME_POPULAR)
│   │   └── 지도자 탭 (INSTRUCTOR_TAB)
│   │       ├── 내 주변 인기 지도자 (HOME_POPULAR_INSTRUCTORS) [NEW]
│   │       │   └── 지도자 상세 (INSTRUCTOR_DETAIL) [NEW]
│   │       └── 지도자 종목 선택 (INSTRUCTOR_SPORTS) [NEW]
│   │           └── 지도자 리스트 (INSTRUCTOR_LIST) [NEW]
│   │               └── 지도자 상세 (INSTRUCTOR_DETAIL) [NEW]
│   ├── 지도 탭 (MAP_TAB)
│   └── 마이페이지 탭 (MYPAGE_TAB)
└── 역할별 전용 (ROLE) - 권한 필요
    ├── 시설 운영자 전용
    ├── 지도자 전용
    └── 관리자 전용
```

## 3. 지도자 관련 기능 정의

### 3.1 홈 화면 지도자 탭 (HOME_INSTRUCTOR_TAB)
**화면 ID**: HOME_INSTRUCTOR_TAB  
**계층 레벨**: 3  
**상위 화면**: 홈 탭  
**하위 화면**: 내주변인기지도자, 지도자종목선택  
**네비게이션 타입**: 탭네비게이션  
**접근 방식**: 홈화면지도자탭클릭  
**사용자 타입**: ALL  

#### 주요 기능
- 홈 화면에서 지도자 탭 선택 시 "내 주변 인기 지도자" 섹션 표시
- 지도자 종목 선택 기능 제공
- 시설 조회와 동일한 UI/UX 구조 적용

#### 하위 기능
- **내 주변 인기 지도자**: 현재 위치 기준 인기 지도자 표시
- **지도자 종목 선택**: 구기, 라켓, 격투/무술, 피트니스, 동계스포츠, 골프/레저, 체대입시&장애인 등
- **지도자 리스트**: 선택한 종목의 지도자 목록 표시
- **지도자 상세**: 선택한 지도자의 상세 정보 조회

#### 상세 설명
홈 화면에서 지도자 탭을 선택하면 "내 주변 인기 지도자" 섹션이 표시됩니다. 시설 탭에서는 "내 주변 인기 시설"이 표시되는 것과 동일한 구조로, 지도자 탭에서는 "내 주변 인기 지도자"가 표시됩니다. 하단의 종목 아이콘들을 통해 해당 종목의 지도자들을 탐색할 수 있습니다.

### 3.2 내 주변 인기 지도자 (HOME_POPULAR_INSTRUCTORS)
**화면 ID**: HOME_POPULAR_INSTRUCTORS  
**계층 레벨**: 4  
**상위 화면**: 홈 화면 지도자 탭  
**하위 화면**: 지도자상세  
**네비게이션 타입**: 카드표시  
**접근 방식**: 지도자카드클릭  
**사용자 타입**: ALL  

#### 주요 기능
- 현재 위치 기준 인기 지도자 표시
- 지도자 카드 형태로 정보 제공
- 거리, 평점, 리뷰 수 등 정보 표시

#### 표시 정보
- **지도자 사진**: 프로필 이미지
- **이름**: 지도자 실명
- **한줄 소개**: 간단한 자기소개
- **거리**: 현재 위치로부터의 거리
- **평점**: 사용자 평점 (별점)
- **리뷰 수**: 후기 개수

#### 상세 설명
현재 위치를 기준으로 한 인기 지도자들을 카드 형태로 보여주는 섹션입니다. 시설 탭의 "내 주변 인기 시설"과 완전히 동일한 구조와 UI/UX로 지도자 정보를 표시합니다. 사용자는 일관된 경험으로 시설과 지도자를 탐색할 수 있습니다.

### 3.3 지도자 종목 선택 (INSTRUCTOR_SPORTS)
**화면 ID**: INSTRUCTOR_SPORTS  
**계층 레벨**: 4  
**상위 화면**: 홈 화면 지도자 탭  
**하위 화면**: 지도자리스트  
**네비게이션 타입**: 버튼입력  
**접근 방식**: 지도자종목버튼선택  
**사용자 타입**: ALL  

#### 주요 기능
- 구기 종목, 라켓 종목, 격투/무술, 피트니스, 동계스포츠, 골프/레저, 체대입시&장애인 등 종목 선택
- 선택한 종목의 지도자 리스트로 이동
- 시설 종목 선택과 동일한 UI/UX

#### 상세 설명
지도자를 종목별로 나누어 사용자가 원하는 종목의 지도자를 선택할 수 있는 기능입니다. 시설의 종목 선택과 동일한 구조로 7개 카테고리로 구성되어 있습니다.

### 3.4 지도자 리스트 (INSTRUCTOR_LIST)
**화면 ID**: INSTRUCTOR_LIST  
**계층 레벨**: 5  
**상위 화면**: 지도자 종목 선택  
**하위 화면**: 지도자상세  
**네비게이션 타입**: 스크롤리스트  
**접근 방식**: 지도자선택  
**사용자 타입**: ALL  

#### 주요 기능
- 선택한 종목에 해당하는 지도자 리스트 표시
- 지도/리스트 전환 가능
- 정렬 및 필터링 기능

#### 표시 정보
- **이름**: 지도자 실명
- **한줄 소개**: 간단한 자기소개

#### 상세 설명
선택한 운동 종목에 해당하는 모든 지도자를 리스트 형태로 보여주는 화면입니다. 거리순, 평점순, 즐겨찾기순 등 다양한 정렬 옵션과 필터링 기능을 제공하며, 지도 뷰와 리스트 뷰 간 전환이 가능합니다.

### 3.5 지도자 상세 (INSTRUCTOR_DETAIL)
**화면 ID**: INSTRUCTOR_DETAIL  
**계층 레벨**: 6  
**상위 화면**: 지도자 리스트 또는 내 주변 인기 지도자  
**하위 화면**: 없음  
**네비게이션 타입**: 화면표시  
**접근 방식**: 지도자클릭  
**사용자 타입**: ALL  

#### 주요 기능
- 지도자 포트폴리오 내용 표시
- 문의하기 버튼 제공

#### 표시 정보

##### 포트폴리오 내용
- **지도자 소개**: 지도자에 대한 상세한 소개
- **전문 분야**: 주로 가르치는 운동 종목들
- **경력 및 자격**: 지도 경력, 자격증
- **활동 사진**: 지도 활동 관련 사진들
- **지도 철학**: 지도 방향 및 철학
- **특별한 경험**: 특별한 경력이나 성과

##### 문의하기
- **문의하기 버튼**: 지도자에게 문의할 수 있는 버튼
- **연락처 정보**: 이메일, 전화번호 등 연락 방법

#### 상세 설명
지도자 리스트에서 선택한 지도자의 포트폴리오 내용을 상세히 볼 수 있는 화면입니다. 지도자의 소개, 전문 분야, 경력, 활동 사진 등 포트폴리오 정보를 제공하고, 문의하기 버튼을 통해 지도자와 연락할 수 있습니다.
