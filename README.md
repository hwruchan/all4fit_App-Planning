## 🎯 PM Core Responsibilities

단순 기획을 넘어 개발 파트너사(CLAPON)와 협업하며 제품의 완성도를 높이는 **Technical PM** 역할을 수행했습니다.

* **Product Strategy**: 1·2차 개발 범위를 확정하고 서비스 로드맵 수립 (v1, v2 Milestone)
* **Technical Specs**: 기능 정의서(PRD), DB 설계서, API 명세서 초안 작성을 통한 개발 가이드 제시
* **UX/UI Planning**: 화면설계서(Wireframe)를 바탕으로 사용자 여정 및 인터랙션 최적화
* **Logic Visualization**: Mermaid Flowchart를 활용하여 복잡한 예약 및 관리자 로직 시각화
* **Risk Management**: 기획-개발 간의 기술적 간극 조율 및 스토어(iOS/Android) 런칭 프로세스 관리
